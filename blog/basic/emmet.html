emmet cheat-sheet

HTML

아래 모든건 입력후 tab키를 눌러야 실행

! 입력

간단하고 빠르게 HTML문서 타입이 출력

div>ul>li

> 를 사용하여 자식 요소를 생성할 수 있다

div>ul+ol+div

+ 를 사용하여 한 요소와 같은 단계에 위치한 요소를 생성할 수 있다

div>li*3 입력

> 를 사용하여 한 단계 위에 요소를 배치할 수 있다

그룹화: div>(header>ul>li*2>a)+footer

() 를 사용해 괄호안에 있는 코드를 묶어 그룹화를 시킨다

div>header>ul>li*2>a+footer

위를 비교해서 그룹화 하지 않을때는 다른 코드가 된다

텍스트 입력: p.container{텍스트넣기}

p태그안에 텍스트가 입력된다

자동 넘버링 부여: p.container{item$}*5

$값에 숫자가 입력된다

#page>#nav>ul>li.item{hello$}*5+dl^dd

응용하면 빠르게 마크업이 가능

CSS

아래 모든건 입력후 tab키를 눌러야 실행

.item{h100+w100}

width, height 기본값은 px단위이며 단위를 교체하는것도 가능

.item{m100p+p100e}

margin, padding

.item{c#111}

color

.item{fsz13}

font-size
