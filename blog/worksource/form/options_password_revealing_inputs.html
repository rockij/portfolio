Classic Technique

Flop out type="password" for type="text" in HTML.

⚠️ Downsides: doesn't play nice with all password managers.

WebKit-Specific Technique

Use type="text" with -webkit-text-security in CSS.

⚠️ Downsides: Not using the sematically correct input type. Doesn't work in all browsers and probably never will.

Modern CSS Technique

Use input-security in CSS.

⚠️ Downsides: Not supported in all browsers yet.
