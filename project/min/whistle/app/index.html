20.11.09 Start Work

NECESSARY

  • 아이폰 12.0 이상 지원 / 안드로이드 9.0 이상 지원
  • 전체동의 및 체크박스 선택 부분은 작업전 개발협의
  • html안 팝업 태그는 id="wrap" 바깥으로 위치
  • 기본폰트 크기 16px (작은폰 14px)
  • SASS로 작업 css는 min로 컴파일러 해서 넘김
  • 서체 : Noto Sans KR(기본) / Nunito Sans(숫자)
  • 입력박스 입력 X 버튼 존재 확인
  • 오픈라이센스,약관 관리자 등록
  • 업무망에선 font.local.css 참조
  • 인터넷망에선 font.cdn.css 참조
  • tab클릭시 한페이지에서 전환이 이뤄질때 tabSliding(1); 추가
  • 감정이모티콘 common.min.css > [feel-emoticon] 활용
  • 개발완료 후 최종 반영할때는 웹폰트로 적용 design.js 상단 확인
