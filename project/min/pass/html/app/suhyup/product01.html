    • 적용금리 : 기본금리 연 0.9%+ 우대금리 0.1%p

    • 입금한도 : 1천만원
    • 입금하실 때 : 딴주머니로 입금하기 위해서는 연결된 잇(it)딴주머니 통장으로부터의 입금만 가능
    • 출금하실 때 : 딴주머니에서 출금하기 위해서는 연결된 잇(it)딴주머니통장으로의 출금만 가능하고 그 외의 출금은 제한. 자동이체, 카드결제계좌 및 현금 출금계좌로 사용 불가

    2. 타행자동이체 수수료 면제
    3. 납부자자동이체 수수료 면제
