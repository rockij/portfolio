Android 4.4미만, iOS 10미만 PASS 앱 업데이트 지원 종료 안내

2020.12.15

안녕하세요! 쉽고 빠른 인증 PASS입니다. PASS 앱을 이용하시는 고객님께 감사의 말씀드리며, Android 4.4미만, iOS 10미만 PASS 앱 업데이트 지원 종료 안내입니다. 2020년 12월 15일부터 사용중인 휴대폰의 OS 버전이 Android 4.4, iOS 10미만일 경우 PASS 앱의 최신 버전 업데이트 지원 종료됩니다. -적용 예정일 2020년 12월 15일 (일정 변경 시 재 공지 드리겠습니다.) -지원 종료 OS 버전 Android 4.4 미만 iOS 10 미만 각 OS 제공사의 기술지원 중단 및 보안 취약 등으로 OS 최소 버전 상향 조정이 불가피한 점 양해 부탁드립니다. 지원 종료 버전에 해당하시는 고객님께서는 휴대폰 OS 업그레이드 후 PASS App을 이용 부탁드리며, 이용을 원치 않으시면 아래의 경로로 PASS 서비스 해지를 진행바랍니다. * OS 버전 및 해지 안내 - OS 버전 확인 1) Android -> 방법1_PASS 앱 > 하단 Navigation 전체 Tab > 고객센터 > Device OS 정보 -> 방법2_설정 > 휴대전화 정보 > 소프트웨어정보 > 안드로이드 버전 2) iOS -> 방법1_PASS 앱 > 하단 Navigation 전체 Tab > 고객센터 > Device OS 정보 -> 방법2_설정 > 일반 > 정보 > 소프트웨어 버전 - PASS 서비스 및 부가서비스 해지 안내 -> PASS 서비스 : PASS 앱 > 전체 > 설정(톱니바퀴모양) > PASS 서비스 해지 -> PASS 부가서비스 : PASS 앱 > 전체 > 설정(톱니바퀴모양) > 서비스 설정/해지관리 PASS 앱 이용이 불가능하신 고객께서는 문자방식으로 휴대폰 본인확인 서비스 이용 부탁드립니다. * 이용경로 - 모바일 : 통신사 선택 다음화면 하단 문자로 인증하기 클릭 - 웹 : 통신사 선택 다음화면 우측 상단 문자인증 클릭 더 자세한 안내와 문의를 원하시는 고객께서는 pass@sk.com로 연락주시기 바랍니다. 고객님께서 더 안전하고 편리하게 PASS를 이용하실 수 있도록 최선의 노력을 다 하겠습니다.
