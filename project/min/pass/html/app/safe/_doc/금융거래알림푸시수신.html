        <아래>
        - SK텔레콤㈜의 PUSH를 통한 금융거래 모니터링 결과 통보
        - 금융거래 모니터링 알림 PUSH 발송을 위해 핀브릿지 주식회사가 금융거래 상세내용을 SK텔레콤㈜에 제공
        -SK텔레콤(주)은 제공받은 정보를 제공일로부터 1개월간 보유/이용

        * 금융거래 모니터링 알림 통보를 원하지 않는 경우 PUSH 수신을 거부할 수 있습니다.
        * PUSH 수신에 동의하였으나, PASS 앱 미설치 또는 삭제 등의 사유로 해당 PUSH를 수신할 수 없는 경우에는 핀브릿지 주식회사가 금융거래 모니터링 알림을 문자메시지(SMS/LMS)로 통보할 수 있습니다.
        * 금융거래 모니터링 알림 PUSH는 무료서비스이며, 금융거래 상세내용 확인은 유료서비스입니다.
