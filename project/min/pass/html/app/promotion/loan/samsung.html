PASS X 삼성카드

PASS에서 삼성카드로 대출 이용 시 이자지원금 1%를 드립니다.

  • 행사기간

    2020.01.01(수)~2020.01.31(금)

  • 행사내용

    행사기간 동안 PASS를 통해 삼성카드의 대출 상품 이용 시 대출실행액의 1%에 해당되는 금액을 지급

  • 혜택제공 방법

    삼성카드에 등록된 휴대폰 번호로 쿠폰번호 발송
    - 해당 쿠폰 번호를 PASS 앱 내 ‘전체’ > ‘쿠폰번호등록’ 메뉴에 쿠폰 번호 입력
    - 쿠폰 번호 입력 후 3일 이내 PASS에 등록된 PASS 주계좌로 입금
    ※ 쿠폰은 2020년 2월말 일괄 발송 예정

유의사항

            - 부적절한 방법을 통해 참여 시 행사대상에서 제외됩니다.
            - 행사기간 내 PASS 앱을 통해 대출을 신청하고 대출금 지급이 완료된 고객이 대상이며, 2월말 이전에 대출금액을 모두 조기상환할 경우 대상에서 제외됩니다.
            - PASS 쿠폰의 이용기간은 발송일 기준 1개월이며, 이용기간이 지난 경우 재발송되지 않으므로, 반드시 기간 내에 PASS 앱에서 쿠폰번호를 입력해주시기 바랍니다.
            - PASS 쿠폰번호 입력 전 이동통신사 변경 등으로 SK텔레콤의 PASS 앱을 이용할 수 없는 경우에는 이자지원금이 지급되지 않습니다.
            - 5만원 초과 이자지원금의 경우 소득세법 등 관련 규정에 따라 22%의 세율(지방소득세 포함)로 원천징수한 소득세 등을 공제 후 지급됩니다.(5만원 이하의 이자지원금은 공제 없이 전액 지급)
            - 5만원 초과 이자지원금을 포함한 기타 소득이 연 300만원이 넘는 분들께서는 직접 관할 세무서에 2020년 5월 31일까지 종합소득과세표준 신고 및 납부를 하셔야 하므로, 구체적인 내용과 절차에 대해서 필요 시 세무 관련 자문을 받으시길 바랍니다.
            - SK텔레콤의 과세 신고 목적 상 5만원 이상 당첨자의 본인 신분증 사본을 요청할 수 있습니다.
            - SK텔레콤은 본 이벤트의 광고만 제공하고 있으며, 각 대출상품에 대한 책임은 없습니다.
