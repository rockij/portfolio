PASS X 씨티은행

PASS에서 씨티은행 대출 이용하면
총 1.0p% 금리할인과 이자지원금 1%도 드립니다!

  • 행사기간

    2020.01.01(수)~2020.01.31(금) (신청일 기준)

  • 행사내용

    행사기간 동안 PASS를 통해 씨티은행의 대출 상품 이용 시
    1. 모바일을 통한 신청 시 0.5%p 금리 우대 혜택 제공
    2. 제휴코드 ‘011011’을 입력하면 0.5%p 금리 추가할인
    3. 대출 실행액의 1%에 해당되는 금액을 지급

  • 혜택제공 방법

    씨티은행에 등록된 휴대폰 번호로 쿠폰번호 발송(2020년 2월말 일괄 발송 예정)
    - 해당 쿠폰 번호를 PASS 앱 내 ‘전체’ > ‘쿠폰번호등록’ 메뉴에 쿠폰 번호 입력
    - 쿠폰 번호 입력 후 3일 이내 PASS에 등록된 PASS 주계좌로 입금

            - 씨티은행 온라인 직장인 신용대출 서비스는 신청 과정에서 고객님께 맞는 최적화된 신용대출 상품을 자동으로 추천해드리며, 최종 추천상품이 새희망홀씨대출인 경우 제휴, 온라인 금리인하 혜택은 적용되지 않습니다.
            - 본 이벤트는 이벤트 기간 내 PASS 앱을 통해 대출을 신청하고 대출금 지급이 완료된 고객이 대상이며, 2월 말 이전에 대출금액을 모두 조기 상환할 경우에 대상에서 제외됩니다.
            - PASS 쿠폰의 이용기간은 발송일 기준 1개월이며, 이용기간이 지난 경우 재발송 되지 않으므로, 반드시 기간 내에 PASS 앱에서 쿠폰번호를 입력 해 주시기 바랍니다.
            - PASS 쿠폰번호 입력 전 이동통신사 변경 등으로 SK텔레콤의 PASS앱을 이용할 수 없는 경우에는 이자 지원금이 지급되지 않습니다.

            - 본 이벤트는 여러 번 참여하시더라도 이자지원금 혜택은 최초 1회만 지급됩니다.
