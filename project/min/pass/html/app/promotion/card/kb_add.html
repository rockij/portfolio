    ※ 쇼핑지원금 쿠폰 번호를 PASS 앱 내 "전체" > "제휴/이벤트" > "쿠폰등록" 메뉴에서 입력 시 3일 이내에 PASS 주계좌로 쇼핑지원금이 입금됩니다.
    ※ 쇼핑지원금을 수령하기 위해서는 PASS 주계좌가 개설되어 있어야 하며, PASS 주계좌가 없을 경우 “금융” > “계좌개설” 메뉴에서 PASS 주계좌를 개설할 수 있습니다.
