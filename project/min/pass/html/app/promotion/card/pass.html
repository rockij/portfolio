PASS X PASS은행

PASS은행에서 대출 이용 시
PASS에서 최대 300만원 이자지원금을 드립니다.

  • 행사기간

    2019.06.03(월) ~ 2019.06.30(일)

  • 행사내용

    행사기간동안 PASS를 통해 PASS은행의 대출 상품 이용 시,
    대출실행액의 3%에 해당되는 이자지원금을 지급(1인 1회, PASS에 등록된 PASS 주계좌로 입금)

  • 혜택제공 방법

    2019.07.10 이후 PASS은행에 등록된 휴대폰 번호로 쿠폰 번호 발송
    해당 쿠폰번호를 PASS 앱 내 ‘전체’ > ‘쿠폰번호 등록’ 메뉴에 입력 시 3일 이내 PASS에 등록 계좌로 입금
