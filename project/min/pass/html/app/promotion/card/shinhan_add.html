              - 행사 기간 안에 일시불 및 할부로 결제한 경우 혜택을 받으실 수 있으며, 일부 결제(쿠폰 이용, 포인트 사용 결제 등) 또는 결제를 취소한 경우 혜택을 받으실 수 없습니다.
              - 신규발급고객(신규/만기/탈회 후 3개월 경과)의 경우 본 PASS 앱 이벤트 페이지를 통해 발급한 고객님만 혜택을 받으실 수 있습니다.
              - 행사 시작일 직전 1년간 신한(신용)카드 이용이력이 없는 고객이라도, 이벤트 대상 카드로 재발급(교체, 추가, 탈회후 재발급)하는 경우 혜택이 제공되지 않습니다.
              - PASS 앱 이벤트 외 신한카드의 다른 이벤트에 이미 참여한 고객은 중복하여 혜택을 받을 수 없으며, 혜택이 먼저 지급되는 이벤트에 한해 1회만 혜택을 받으실 수 있습니다.(카드발급 캐시백 이벤트, 친구추천 이벤트 등과 중복적용 불가)

              2.가족카드를 갖고 계신 고객님이 본인 카드를 발급한 경우
              3.카드 해지 또는 신한카드 회원 탈퇴한 경우
