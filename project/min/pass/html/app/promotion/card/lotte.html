PASS X 롯데카드

PASS에서 발급받은 롯데카드로 첫 결제하면 쇼핑지원금 5만원을 드립니다.

  • 행사기간

    2019.12.01(일) ~ 2019.12.31(화)

  • 대상고객

    PASS 앱에서 이벤트 대상 카드를 발급받은 고객
    단, 행사시작일 기준 직전 12개월 동안(2018.12.01~2019.11.30) 롯데카드 결제이력이 없는 고객에 한함

  • 행사내용

    EVENT 1. 행사기간 중 대상카드로 첫 결제 시 쇼핑지원금 5만원 지급
    EVENT 2. 행사 기간 중 PASS 앱을 통해 발급받은 롯데카드 신규회원 연회비 100% 캐시백
    (단, 롯데카드 I'm YOLO는 연회비 캐시백 제외)

  • 혜택제공 방법

    EVENT 1. 대상 고객에 대해 롯데카드에 등록된 휴대폰 번호로 쇼핑지원금 쿠폰 번호 발송
    EVENT 2. 롯데카드에 등록된 결제 계좌로 연회비 캐시백 입금

    ※ 쇼핑지원금 쿠폰 번호를 PASS 앱 내 "전체" > "제휴/이벤트" > "쿠폰등록" 메뉴에서 입력 시 3일 이내에 PASS 주계좌로 쇼핑지원금이 입금됩니다.
    ※ 쇼핑지원금을 수령하기 위해서는 PASS 주계좌가 개설되어 있어야 하며, PASS 주계좌가 없을 경우 "금융" > "계좌개설" 메뉴에서 PASS 주계좌를 개설할 수 있습니다.
    ※ 자세한 내용은 유의사항을 확인해주시기 바랍니다.

  • 문자 발송 예정일

    2020. 01. 31(금)

유의사항

            EVENT 1.
            - 행사 직전 12개월 동안 롯데카드 본인 및 가족카드 결제이력이 있는 고객은 이벤트 대상에서 제외됩니다.
            - 행사 기간 중 이벤트 대상이 아닌 롯데카드로 첫 결제 발생 시 이벤트 대상에서 제외됩니다.
            - 이벤트 기간 내 대상카드(8종)에 대해 중복 참여가 불가합니다.
            (중복 시 우선 결제 건으로 참여)
            - 첫 결제는 행사 기간 내 이벤트 대상 카드로 결제 금액 한도 없이 1회 이상 결제 시 대상이며, 상품권 업종 결제 금액은 제외입니다.
            - 기존에 보유한 이벤트 대상 카드 해지 후 재발급 참여 시 대상이 아닙니다.
            - 이벤트 기간 내 전표 미매입, 승인거절, 승인취소, 부분취소, 환급등의 사유로 정상처리 되지 않을 시 대상이 아닙니다.
            - PASS 쿠폰 지급 시점에 카드 해지, 교체, 탈회 및 PASS 탈퇴 시 적립이 불가합니다.
            - 쿠폰 사용 문의는 PASS 고객센터(tpasshelp@gmail.com), 이벤트 문의는 롯데카드 고객센터(1588-8100)로 연락 바랍니다.
            - 이벤트 유의사항 미확인 사유로 인한 적립금 예외 지급은 처리 불가합니다.
            - 결제는 2020년 1월 7일(화)까지 승인된 건까지만 포함됩니다.
            - 이 행사는 신규 연회비 캐시백 행사와 함께 진행됩니다. 대상 카드 및 상세 내용은 카드 발급 페이지의 안내를 확인해주시기 바랍니다.
            - 쇼핑지원금 쿠폰은 발송일 기준 1개월 안에만 등록하실 수 있습니다. 이용 기간이 지난 경우 다시 발급되지 않으므로 반드시 기간 안에 PASS 앱에서 쿠폰 번호를 입력해주시기 바랍니다.
            - 쇼핑지원금을 수령하기 위해 PASS 앱에서 쿠폰번호를 입력하기 위해서는 PASS 앱 내에 PASS 주계좌(신한금융투자 CMA)가 개설되어 있어야 하며, PASS 주계좌가 개설되어 있지 않은 경우 계좌 개설을 요청할 수 있습니다.
            - PASS 주계좌는 "금융" > "계좌개설" 메뉴에서 개설할 수 있습니다.
            - PASS 주계좌를 개설하지 않거나, PASS 쿠폰 번호를 입력 전 이동통신사 변경 등으로 SK텔레콤의 PASS 앱을 이용할 수 없는 경우 쇼핑지원금이 지급되지 않습니다.
            - 이벤트 기간, 내용 및 적립예정일은 롯데카드, SK텔레콤의 사정에 따라 변경 또는 조기종료 될수 있습니다.
            - 카드 발급 신청시 상품설명서 및 상품약관을 반드시 확인하기시 바랍니다.
            - 자세한 내용은 롯데카드 홈페이지를 참고하시거나 
              고객센터(1588-8100)로 문의 바랍니다.

            EVENT 2.
            - 본인명의 개인 신용카드를 처음 발급받는 신규회원만 혜택이 제공됩니다.
            - 가족카드 발급 건 및 가족카드를 이미 소지한 회원님의 본인카드 발급 건은 이벤트 대상에서 제외됩니다.
            - 롯데카드의 다른 발급 이벤트 혜택과 중복 적용되지 않으며, 심사기준에 따라 발급이 제한될 수 있습니다.
            - 이 이벤트페이지에서 카드 신청한 회원만 혜택 적용됩니다.
