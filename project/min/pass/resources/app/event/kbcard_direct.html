KB국민카드 자동납부로 편리함과 혜택 모두 챙기세요!
대상: KB국민카드 회원(KB국민 기업, 비씨, 선불카드 제외)
참여방법: ‘자동납부 신청하기’ 버튼을 통해 신청
혜택지급: 캐시백-납부 다음달 20일 이내 카드 결제계좌로 입금예정

    최초납부시 캐시백 5천원

유의 사항
  • 캐시백은 KB국민카드로 항목별 자동납부 최초 납부 시에만 한해 회원별 (가족카드 포함) 1회 지급됩니다.
  • 지급 이전 자동납부 또는 카드 해지 시 대상에서 제외됩니다.
  • 쇼핑지원금 쿠폰은 발송일 기준 익월 말까지만 등록하실 수 있습니다. 등록 기간이 지난 경우 다시 발급되지 않으므로 반드시 기간 안에 PASS앱에서 쿠폰 번호를 입력하여 주시기 바랍니다.
  • PASS 머니의 경우 2021년 1월 1일 이후 자동납부 신규고객 기준으로 제공됩니다.
  • PASS 머니는 행사기간 중 매월 1일부터 말일까지 등록완료 기준으로 혜택이 지급됩니다.
  • PASS 머니 혜택은 항목별 1회만 지급(아파트 2건, 도시가스 1건 신청 시 아파트 및 도시가스 각 1건에 대한 혜택 제공)됩니다.
  • PASS 머니 혜택은 고객별 최초 1회만 지급되므로 자동납부 신청시 같은달에 신청 바랍니다.
    (예시: 1월 아파트관리비 등록 후 2월 도시가스 등록 시 1월 등록한 아파트관리비에 대한 혜택만 지급)
  • PASS 앱 내 “전체” > “제휴/이벤트” > “쿠폰등록” 메뉴에서 쇼핑지원금 쿠폰 번호를 입력하면 3일 이내에 PASS 머니로 쇼핑지원금이 지급됩니다.
  • PASS 머니로 쇼핑지원금을 수령한 후 본인 명의의 계좌로 송금할 수 있습니다.
  • 쇼핑지원금 쿠폰 번호 입력 전 이동통신사 변경 등으로 SK텔레콤의 PASS 앱을 이용할 수 없는 경우 쇼핑지원금이 지급되지 않습니다.
  • 쿠폰과 관련된 문의는 SKT PASS 고객센터(pass@sk.com)로 연락해 주시기 바랍니다.
  • 본 행사는카드사 및 제휴사 사정으로 사전고지 없이 중단되거나 변경될 수 있습니다.
