모바일운전면허 확인서비스란?

내 운전면허증을 PASS 앱에 등록해 모바일로 사용하는
서비스로, ICT 규제샌드박스 승인을 받아 운전면허증과 동일한 법적 효력이 있습니다.

이벤트 기간

2020년 10월 5일(월) ~ 2020년 11월 6일(금)

이벤트 경품
  • 추천왕 1등(1명): 1백만 원 *제세공과금 별도
  • 추천왕 2~5등(4명): 50만 원 *제세공과금 별도
  • 추천왕 6~15등(10명): 5만 원
  • 참가상: 1명만 추천해도 100% 당첨 PASS 머니 랜덤 박스 지급! (참가상 경품은 일찍 소진될 수 있으며, 소진 시 경품이 지급되지 않습니다.)
  • 추천 코드 입력하고 신규가입하신 고객님(100명): 스타벅스 카페 아메리카노(Tall) 1잔 모바일 상품권
당첨자 발표

PASS 머니 랜덤 박스(참여 즉시 지급) 외 경품 당첨자는 11월 중 개별 안내

유의사항
  • 이 이벤트는 PASS 앱에 운전면허증을 등록하신 고객님만 참여하실 수 있습니다.
    운전면허증 등록을 하지 않으신 고객님은 추첨 대상에서 제외됩니다.
  • 추천왕 경품은 PASS 머니로 지급되며, 지급된 PASS 머니는 즉시 고객님의 일반 계좌로 이체하실 수 있습니다.
  • PASS 머니를 받으려면 당첨 후 기한 안에 PASS 머니에 가입(무료)하셔야 합니다. PASS 머니에 가입하지 않으면 당첨이 취소될 수 있습니다.
  • 스타벅스 모바일 상품권은 추천 코드를 입력하고 신규가입하신 고객님을 대상으로 추첨을 통해 증정하며, 탈퇴 후 다시 가입해 이벤트에 참여한 경우 추첨 대상에서 제외됩니다.
  • 이벤트에 참여한 뒤 휴대폰 번호 변경, 명의 변경, 회선 해지, 서비스 해지를 한 경우 당첨이 취소될 수 있습니다.
  • 이 이벤트는 참여사의 사정에 따라 별도의 안내 없이 중단 또는 변경될 수 있습니다.
  • 5만 원을 초과하는 경품은 당첨 금액에서 제세공과금(22%) 공제 후 지급됩니다.
  • 문의: pass@sk.com
