비밀번호 재설정

회원님의 계정 비밀번호를 재 설정해주세요.

  • 비밀번호는 6~15자로 입력 해주세요.
  • 비밀번호는 영문, 숫자, 특수문자 중 2가지 이상을 혼합하여 입력 해주세요.
  • 아이디와 동일한 비밀번호는 사용이 불가 합니다.
  • 비밀번호에 공백은 사용할 수 없습니다.
  • 쉬운 비밀번호나 자주 쓰는 사이트의 비밀번호의 경우, 도용되기 쉬우니 주기적으로 변경해주세요.
