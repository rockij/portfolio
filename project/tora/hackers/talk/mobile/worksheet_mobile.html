  • 파일 최상단에 있는 DOCTYPE은 절대로 삭제하시면 안됩니다. 아이프레임을 사용하실 경우 그 안에도 DOCTYPE은 포함시켜 주셔야 합니다.
  • br, input, img 등의 태그는 꼭 마지막이 닫혀선 안됩니다. 예) <br> <input type="text" class="input"> <img src="btn.gif" title="버튼">
  • 태그는 반드시 짝이 맞아야 합니다. 특히 div 닫기가 안 되어 있거나 하는 경우 레이아웃이 크게 어긋나니 태그의 짝 맞춤을 반드시 확인해주세요.
  • form의 각 입력폼(text, select, radio, checkbox, password)에 id값과 label 태그의 for 속성의 값과 같아야 label 의 텍스트를 클릭시 해당 입력폼으로 커서가 이동하게 됩니다.
  • javascript를 사용할 경우 <script type="text/javascript"> 과 같이 꼭 type를 함께 적어주셔야 합니다.
  • 가급적 태그에 직접 스타일을 입히는 것은 피해주시기 바랍니다.
  • jquery 사용시 jquery 관련 js 불러온 후 마지막에 퍼블리시 js(예:base.js)를 불러와야 에러가 나지 않습니다.

※ css, flash, js, images 파일의 수정사항은 별도로 기술하지 않습니다. 수시로 업데이트 해주세요.
