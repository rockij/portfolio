유의사항
  • 가상계좌 건별로 고객정보를 등록하거나 입금조건을 설정하실 수 있습니다.
  • 고객명은 가상계좌로 입금 시 보여지는 해당 가상계좌 수취인명이며, 필수 선택사항입니다.
  • 고객명은 착오송금을 방지하기 위하여 가상계좌를 발급한 업체명이 자동으로 표시되며, 필요 시 업체명 추가입력이 가능합니다.
    예) 업체명-"(주)국민상사", 추가입력-"홍길동"인 경우, 가상계좌 수치인명은 "(주)국민상사_홍길동"으로 보여집니다.
  • 추가입력을 포함한 전체 고객명(한글 15자, 영문/숫자 30자 이내)에서 자동생성된 업체명(한글 8자, 영문/숫자 16자 이내) 자리수를 제외한 추가입력이 가능합니다.
  • 등록고유번호는 선택사항이며, 가상계좌 입금 발생 시 등록고유번호에 입력하신 내용으로 입금자를 확인 할 수 있습니다.
    (주민등록번호 입력 불가하며, 한글은 8자, 영문/숫자는 16자 이내 가능)
    예) 등록고유번호에 "홍길동"이라 입력 후 해당 계좌로 입금이 되면, 등록고유번호 "홍길동"으로 입금내역 확인 가능합니다.
  • 입금조건설정을 하시려면 아래 입금조건설정에 '적용'을 클릭하십시오.
