유의사항
  • 가상계좌별 정보를 대량으로 등록/변경/초기화 하실 수 있습니다.
  • [샘플파일 다운로드]를 클릭해 파일을 다운로드 하신 후, 등록할 정보를 편집하십시오.
  • [파일열기]로 등록할 엑셀파일을 선택합니다.
  • [처리구분]에서 업무를 선택한 후 [파일등록]을 클릭하시기 바랍니다.
  • [계좌정보 초기화]를 하시는 경우, 기존 가상계좌번호에 입력하신 정보는 삭제되므로 유의하시기 바랍니다.
  • 회원사ID와 고객명 정보가 등록된 정보와 일치하지 않을 경우, 계좌정보 등록이 실행되지 않습니다.
  • 착오송금을 방지하기 위하여 가상계좌의 고객명(입금 시 보여지는 가상계좌 수취인명)은 가상계좌 발급업체명이 반드시 포함되어야하며,
    업체명이 미포함된 가상계좌 고객명으로 계속 거래 시 가상계좌 서비스 이용이 중지될 수 있습니다.
파일작성 방법
  • 고객명 (파일 작성 시 본인의 고객명)
    - 업체명만 표시
    - 업체명 + 추가입력
    -
  • 가상계좌로 입금시 보여지는 가상계좌 수취인명 등록을 위한 고객명이며 업체명만 표시 또는 업체명+추가입력 선택이 가능합니다.
    (추가입력을 포함한 전체 고객명은 한글 15자, 영문/숫자 30자 이내 가능)
  • 파일 등록 시 고객명은 자동 생선된 고객명과 반드시 일치하여야 하며, 기 등록된 고객명이 업체명을 포함하지 않았을 경우에는 자동생성된 고객명을 포함하여
    변경등록 하여야 합니다.

입력사항

  • 행의 처리구분이 계좌정보신규등록일 경우 가상계좌를 입력할 수 없습니다.
  • 행의 회원사ID가 없습니다. 회원사ID를 입력해주십시오.
  • 행의 회원사ID에는 특수문자를 입력할 수 없습니다. 회원사ID를 확인해 주십시오.
  • 행의 회원사ID는 5자리입니다. 회원사ID를 확인해 주세요.
  • 고객정보와 입금조건설정 모두 수정을 원하시는 경우
    처리구분 [고객정보수정+입금조건설정]을 선택하세요.
  • 처리구분이 [계좌정보초기화]인 경우 가상계좌번호, 정액금액 0를 입력하세요.
  • 행의 고객명에 업체명+추가입력인 경우, [(한)빈수빈반배] 입력해 주십시오.
  • 행의 고객명에 업체명만 표시할 경우, [(한)빈수빈반배] 입력해 주십시오.
