레이어 팝업 가이드

열기 : Jquery click 이벤트를 이용한 예

열기 : target-layer-open의 속성값에 "열고 싶은 타겟레이어의 클래스명; dim의 사용여부" 를 순서대로 넣어준다.
ex) <a href="javascript:;" target-layer-open="layer;dim;">jquery click open</a>

jquery click open

열기 : javascript function을 이용한 예

uiCommon.layer.open 호출 할때 스트링 파라미터 값으로 타겟레이어의 클래스와 dim 사용 여부를 순서대로 넣어준다
ex) <a href="javascript: uiCommon.layer.open('layer', 'dim');">javascript function open</a>

javascript function open

닫기 : Jquery click 이벤트를 이용한 예

target-layer-close의 속성값에 "닫고 싶은 타겟레이어의 클래스명" 을 넣어준다.
ex) <a href="javascript:;" target-layer-close="layer">jquery click close</a>

닫기 : javascript function을 이용한 예

uiCommon.layer.close 호출 할때 스트링 파라미터 값으로 타겟레이어의 클래스를 넣어준다
ex) <a href="javascript: uiCommon.layer.close('layer');">javascript function close</a>

TAB

placeholder 가이드

input placeholder

IE9의 크로스 브라우징을 위해 placeholder 기능을 스크립트로 구현
input의 data-placeholder 속성 안에 placeholder의 내용을 입력하면 해당 input의 value값에 들어가는 방식.
ex) <input type="text" value="" data-placeholder="입력해주세요.">
